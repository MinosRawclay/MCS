Detailed Description

Declarations of client-server dialogue functions.

This header file declares dialogue functions to manage communications between clients and servers via sockets for the Belote card game. It provides function prototypes for dialogue operations for:

  • The server (accepting and processing connections)
  • The client (communication with the server)
  • Registration (registration procedure)
