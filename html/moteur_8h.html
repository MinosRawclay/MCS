Detailed Description

Header file for the Belote card game implementation.

This header defines the data structures, enumerations, and function prototypes for implementing the French card game Belote for 4 players in 2 teams.

Author
Raphael ALLO
Date
02/02/2026

Enumeration Type Documentation

◆ card

enum card

Enumeration of all cards in the deck.

Cards are organized by suit: Hearts (H), Clubs (C), Spades (P), Diamonds (T) Each suit contains: AS, 7, 8, 9, 10, V (Jack), D (Queen), R (King)

Enumerator
NOTHING 

Represents no card or empty slot.
