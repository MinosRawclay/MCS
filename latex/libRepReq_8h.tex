\doxysubsection{Detailed Description}
Declarations of request/response management structures and functions. 

This header file defines data structures for requests and responses in the communication protocol, as well as formatting macros for conversion between structures and strings. It also declares the complete enumeration of request, response and communication codes for the Belote card game. \begin{DoxyAuthor}{Author}
