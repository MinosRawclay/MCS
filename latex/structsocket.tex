\doxysubsection{Detailed Description}
Socket data structure definition. 

This structure encapsulates all information related to a socket, including its file descriptor, connection mode, and both local and remote address information. It provides a complete representation of a socket for both client and server applications.

\begin{DoxyNote}{Note}
This type is composed of the socket file descriptor, mode (connected/non-\/connected), and application addresses (local/remote) 
\end{DoxyNote}
